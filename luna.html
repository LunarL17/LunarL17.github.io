Game Engine

A muito tempo fiquei trocando de game engines para me decidir de qual usar.

Varias engines tem uma licença muito cara e para um indie isso pode ser um grande problema.

Então decidi criar a minha propria game engine e usar para o meu proximo jogo "AAA". Essa game engine está sendo feita em C# e na base do XNA (Uma framework antiga para jogos).

Essa Game Engine não será pública, pois quero uma coisa que seja apenas meu e qualquer tipo de comercio / parceria para utilizar essa engine sera recusado.

Meu maior proposito de ter criado ela foi para poder treinar minhas habilidades em C# e futuramente meu jogo "AAA" sera feito apartir dela!
